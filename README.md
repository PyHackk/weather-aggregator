# 🌤️ Weather Data Aggregator

Multi-source weather API with intelligent caching and historical tracking.

## Features

- ✅ Fetch weather from OpenWeatherMap & WeatherAPI
- ✅ Redis caching (10-min TTL)
- ✅ Rate limiting (100 req/hour per IP)
- ✅ Historical weather tracking
- ✅ Compare multiple weather sources
- ✅ Swagger API documentation
- ✅ Docker deployment ready

## Tech Stack

- **Backend:** Flask + Flask-RESTX
- **Database:** PostgreSQL
- **Cache:** Redis
- **APIs:** OpenWeatherMap, WeatherAPI
